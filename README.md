# BULL//RUN

A neon 3D endless runner built with Three.js. Control a neon-black bull through a futuristic city, collect cyan energy cores, jump obstacles, and charge-dash through barriers.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by the dev server. Use left/right (or A/D) to change lanes, up/W to jump, and Space to charge-dash.

## Production build

```bash
npm run build
```

The project uses Vinext to produce the host-compatible output in `dist/`.
